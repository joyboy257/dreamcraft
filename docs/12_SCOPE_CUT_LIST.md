# Scope Cut List

## Never on the critical path

- Multiplayer/network synchronization
- User authentication
- Persistent social gallery
- Crafting recipes
- Large inventory/economy
- Infinite terrain
- General combat system
- Procedural dungeons at arbitrary scale
- Full rigid-body world
- Ragdolls, cloth, rope, fluid simulation
- Real-time LLM NPC chat
- Generated JavaScript execution
- Generated shaders
- Image-generated textures/models
- Per-dream builds/deployments
- Native Swift/SceneKit/Unity client
- App Store submission
- VR

## Stretch only after G7

- Capacitor iOS wrapper
- Shareable server-stored dream URLs
- Voice input
- Screenshot/postcard export
- Public gallery
- Additional body plans/features
- More dynamic props
- Advanced greedy meshing/occlusion
- Multiple simultaneous objectives
- Real-time spectator mode
- Optional image-generated mood board before entry

## Cut order under schedule pressure

Cut in this order:

1. Native/Capacitor packaging
2. Public share/gallery persistence
3. Advanced mobile controls
4. Director-parallel runtime strategy
5. Dynamic props
6. Advanced materials/shadows
7. Greedy meshing if exposed-face version performs adequately
8. Extra body plans/behaviors
9. Optional decorative enrichment

Do not cut:

- Safe validated DreamSpec
- Local fallback
- Three semantic anchors
- Readable hero entity
- One objective and ending
- First-person movement/collision
- Reliable public judged path
